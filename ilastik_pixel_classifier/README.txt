To use this project file from the ilastik GUI:- Open project file in ilastik- Go to tab `5. Batch Processing` and open åSelect Raw Data Filesèí to load the images to be processed.- Run the pixel classification using åProcess all filesí, it will generate a new directory called åilastik_outputí in the same directory where the project file is and put there all the resulting probability images.- You can ignore tabs `1. Input Data`, `2. Feature Selection`, `3. Training` and  `4. Prediction Export`, unless you want to make changes to the pixel classifier or the exported data.
